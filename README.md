# React + TypeScript + Vite

- 백오피스 특성상 SEO가 중요하지 않고
- 앱의 규모 또한 비교적 작기 때문에
- NextJS 정도의 프레임워크는 필요없다

위와 같은 배경으로 Stack을 선정했습니다.

# Local Development

[Download Nodejs LTS >= v22](https://nodejs.org/en/)

```bash
npm install

npm run dev
```

port번호를 특정하지 않았다면, 브라우저를 열어 다음 URL을 입력하세요: `http://localhost:5173/`
